- **工业数据与通讯**：OPC UA、MQTT 协议基础，SQL Server / MySQL 基础操作，Python 数据采集。

- **AI 辅助开发**：将 Claude Code / Cursor 等 AI 编程工具融入日常开发流程，在代码编写、调试、重构与技术调研环节实现效率倍增。

- **语言能力**：日语 N2，英语四级；具备对日制造业实习经验，可胜任跨语言技术沟通。
